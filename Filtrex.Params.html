    Filtrex v0.3.0 Filtrex.Params

    Filtrex.Params is a module that parses parameters similar to Phoenix, such as:

    %{"due_date_between" => %{"start" => "2016-03-10", "end" => "2016-03-20"}, "text_column" => "Buy milk"}
